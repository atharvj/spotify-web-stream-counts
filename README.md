# Spotify Web Stream Counts

[![Greasy Fork installs](https://img.shields.io/greasyfork/dt/584364?label=Greasy%20Fork\&labelColor=555\&color=8B0000)](https://greasyfork.org/en/scripts/584364)
[![Version](https://img.shields.io/greasyfork/v/584364?label=Version\&labelColor=555\&color=8B0000)](https://greasyfork.org/en/scripts/584364)

A userscript that adds song stream counts to artist and album pages in the Spotify Web Player.

## How to install

1. Install Tampermonkey or Violentmonkey.
2. Open the [Spotify Web Stream Counts page on GreasyFork](https://greasyfork.org/en/scripts/584364).
3. Click **Install this script**.
4. Confirm the installation in your userscript manager.
5. Open or reload the Spotify Web Player.
